## Survey

- 4~7번 선택지 총 합쳐서 12개 이상 선택해야함

1. 현재 귀하는 어느 분야에 종사하고 계십니까?
   - 일 경험 없음
2. 현재 귀하는 학생입니까?
   - 아니오
3. 현재 귀하는 어디에 살고 계십니까?
   - 가족
4. 귀하는 여가 활동으로 주로 무엇을 하십니까?( 두 개 이상)
   1. 공연보기, 콘서트보기
   2. 영화보기
   3. 공원가기, 해변가기
   4. 주거 개선
5. 귀하의 취미나 관심사는 무엇입니까? ( 한 개 이상)
   1.  요리하기
   2. 음악 감상하기, 악기 연주하기
6. 귀하는 주로 어떤 운동을 즐기십니까? (한 개 이상)
   - 자전거, 조깅, 걷기, 하이킹/트레킹
7. 귀하는 어떤 휴가나 출장을 다녀온 경험이 있습니까?(한 개 이상)
   1. 집에서 보내는 휴가
   2. 국내여행, 해외여행
